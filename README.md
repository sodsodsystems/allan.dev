## 🚀 Quick Start

### View Locally
Simply open `allan.dev.html` in your browser.

### Host on GitHub Pages

1. Create a new repository named `sodsodsystems.github.io`
2. Push this code to GitHub:
   ```bash
   git remote add origin https://github.com/sodsodsystems/sodsodsystems.github.io.git
   git branch -M main
   git push -u origin main
   ```
3. Your site will be live at: `https://sodsodsystems.github.io`

### Or use a custom domain

1. Create a repository named `allan.dev`
2. Add CNAME file with your domain
3. Update DNS records to point to GitHub Pages

## 🎨 Customization

The website uses CSS variables for easy theming. Edit the `:root` section in the `<style>` tag to customize colors:

```css
:root {
  --bg: #0a0a0f;
  --blue: #4f8ef7;
  --purple: #9b5de5;
  --pink: #f72585;
  --text: #e8e8f0;
}
```
